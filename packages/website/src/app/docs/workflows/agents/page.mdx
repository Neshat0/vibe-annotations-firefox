# Annotating with Agents

Let browser-capable AI agents annotate for you, collaboratively or autonomously.

## Compatible Agent Pattern

Any browser-capable agent can use this workflow if it can execute JavaScript on your active page.

## Getting Started

Copy this prompt and paste it into your agent chat:

```
Read window.__vibeAnnotations.help() and use this extension for my comments on this project.
```

The agent discovers the bridge API, then uses `createStyleAnnotation` for broad CSS changes and `createAnnotation` for single-element edits. Changes preview live in the browser and get recorded as annotations.

## Requirements

The extension must be active on the page for the bridge API to be available.

Agents launched in separate browsers (Playwright, Puppeteer, remote sessions) do not load your local extension by default. If needed, launch them with your extension path configured.
